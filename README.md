# 💬 AI Review Responder

Веб-инструмент для генерации профессиональных ответов на отзывы клиентов.

## 🚀 Возможности

- Генерация ответов в 1 клик
- Режим "Ручной" (официальный / дружелюбный / короткий)
- Режим "Авто" (определяет тональность)
- Сохранение истории в файл
- Экспорт ответа в .txt

## 🛠 Технологии

- Python
- OpenAI API
- Streamlit

## ⚙️ Установка

```bash
git clone https://github.com/your-username/ai-review-responder
cd ai-review-responder
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
